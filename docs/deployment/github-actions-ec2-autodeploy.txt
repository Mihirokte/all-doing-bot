--------------------------------------------------------------------------------
SECURITY GROUP REMINDER
--------------------------------------------------------------------------------
  GitHub Actions runners use changing public IPs. For auto-deploy via SSH you need
  either:
    - Security group inbound SSH (22) from 0.0.0.0/0 (simple, weaker), OR
    - A tighter approach: only workflow_dispatch (delete EC2_AUTO_DEPLOY) and run
      deploy when your IP can SSH, OR use AWS SSM / self-hosted runner instead of SSH.
